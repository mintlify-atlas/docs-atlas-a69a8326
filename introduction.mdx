## What is MiroFish?

MiroFish is a next-generation AI prediction engine powered by multi-agent technology. By extracting seed information from the real world (such as breaking news, policy drafts, or financial signals), it automatically constructs a high-fidelity parallel digital world where thousands of intelligent agents interact and evolve.

<Note>
**Powered by OASIS**: MiroFish's simulation engine is powered by [OASIS (Open Agent Social Interaction Simulations)](https://github.com/camel-ai/oasis) from the CAMEL-AI team.
</Note>

## How It Works

MiroFish operates through a five-stage workflow:

<Steps>
  <Step title="Graph Building">
    Extract seed information from real-world sources and construct a GraphRAG (Graph Retrieval-Augmented Generation) structure with individual and collective memory injection.
  </Step>
  
  <Step title="Environment Setup">
    Perform entity relationship extraction, generate agent personas, and inject simulation parameters through environment configuration agents.
  </Step>
  
  <Step title="Simulation Execution">
    Run dual-platform parallel simulations that automatically parse prediction requirements and dynamically update temporal memory.
  </Step>
  
  <Step title="Report Generation">
    The ReportAgent uses a rich toolset to deeply interact with the post-simulation environment and generate comprehensive prediction reports.
  </Step>
  
  <Step title="Deep Interaction">
    Engage in conversations with any agent in the simulated world or interact directly with the ReportAgent for insights.
  </Step>
</Steps>

## Use Cases

### Macro-Level: Decision-Making Laboratory

MiroFish serves as a rehearsal laboratory for decision-makers, enabling:

- **Policy Testing**: Test government policies in a zero-risk environment before implementation
- **Public Relations**: Simulate public opinion responses to different communication strategies
- **Financial Prediction**: Model market reactions to economic events and financial signals
- **Crisis Management**: Rehearse responses to potential crisis scenarios

### Micro-Level: Creative Sandbox

For individual users, MiroFish offers:

- **Story Development**: Predict novel endings and explore narrative possibilities (e.g., predicting the lost ending of "Dream of the Red Chamber")
- **Scenario Exploration**: Experiment with imaginative "what if" scenarios
- **Trend Analysis**: Understand how social trends might evolve
- **Personal Decision-Making**: Simulate outcomes of personal choices

## What You Need

To get started with MiroFish, you'll need:

1. **Seed Materials**: Upload documents such as data analysis reports, news articles, or story texts
2. **Prediction Requirements**: Describe what you want to predict in natural language
3. **API Access**: Configure LLM API (supports any OpenAI SDK-compatible API) and Zep Cloud for memory management

<Note>
MiroFish returns a detailed prediction report and a deeply interactive high-fidelity digital world that you can explore and query.
</Note>
