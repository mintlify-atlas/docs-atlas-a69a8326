## What is MiroFish?

MiroFish is a next-generation AI prediction engine that uses multi-agent technology to create parallel digital worlds. Upload documents containing real-world information, and watch as thousands of intelligent agents interact, evolve, and reveal future trajectories.

## Quick Start

<Steps>
  <Step title="Install Dependencies">
    Clone the repository and install frontend and backend dependencies
    
    ```bash
    git clone https://github.com/666ghj/MiroFish.git
    cd MiroFish
    npm run setup:all
    ```
  </Step>
  
  <Step title="Configure Environment">
    Set up your LLM API keys and Zep Cloud credentials
    
    ```bash
    cp .env.example .env
    # Edit .env with your API keys
    ```
  </Step>
  
  <Step title="Start the Application">
    Launch both frontend and backend services
    
    ```bash
    npm run dev
    ```
    
    Access the application at `http://localhost:3000`
  </Step>
</Steps>

## Use Cases

MiroFish enables powerful prediction scenarios across domains:

- **Public Opinion Analysis**: Predict how social sentiment evolves around policy changes or news events
- **Financial Forecasting**: Simulate market reactions to economic signals and corporate announcements  
- **Scenario Planning**: Test strategic decisions in risk-free digital environments before real-world execution
- **Creative Exploration**: Predict alternative storylines, plot developments, or narrative outcomes
