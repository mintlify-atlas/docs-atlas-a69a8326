## Get Report by ID

### GET /api/report/{report_id}

Retrieve complete report details including metadata, outline, and full Markdown content.

**Path Parameters**

<ParamField path="report_id" type="string" required>
  The report ID (format: `report_xxxxxxxxxxxx`)
</ParamField>

**Response**

<ResponseField name="success" type="boolean">
  Whether the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <ResponseField name="report_id" type="string">
    The report ID
  </ResponseField>
  
  <ResponseField name="simulation_id" type="string">
    Associated simulation ID
  </ResponseField>
  
  <ResponseField name="status" type="string">
    Report status: `pending`, `planning`, `generating`, `completed`, or `failed`
  </ResponseField>
  
  <ResponseField name="outline" type="object">
    Report structure
    
    <ResponseField name="title" type="string">
      Report title
    </ResponseField>
    
    <ResponseField name="summary" type="string">
      One-sentence executive summary
    </ResponseField>
    
    <ResponseField name="sections" type="array">
      Array of sections
      
      <ResponseField name="title" type="string">
        Section title
      </ResponseField>
      
      <ResponseField name="content" type="string">
        Section Markdown content
      </ResponseField>
    </ResponseField>
  </ResponseField>
  
  <ResponseField name="markdown_content" type="string">
    Complete report in Markdown format
  </ResponseField>
  
  <ResponseField name="created_at" type="string">
    ISO 8601 creation timestamp
  </ResponseField>
  
  <ResponseField name="completed_at" type="string">
    ISO 8601 completion timestamp (null if not completed)
  </ResponseField>
  
  <ResponseField name="error" type="string">
    Error message if status is `failed` (null otherwise)
  </ResponseField>
</ResponseField>

<RequestExample>
```bash cURL
curl https://api.mirofish.com/api/report/report_f8a3c91b2e4d
```

```python Python
import requests

response = requests.get(
    "https://api.mirofish.com/api/report/report_f8a3c91b2e4d"
)

report = response.json()["data"]
print(f"Status: {report['status']}")
print(f"Title: {report['outline']['title']}")
print(f"Sections: {len(report['outline']['sections'])}")
```
</RequestExample>

## Get Report by Simulation

### GET /api/report/by-simulation/{simulation_id}

Retrieve the report associated with a specific simulation.

**Path Parameters**

<ParamField path="simulation_id" type="string" required>
  The simulation ID
</ParamField>

**Response**

Same structure as GET /api/report/{report_id}, plus:

<ResponseField name="has_report" type="boolean">
  Whether a report exists for this simulation
</ResponseField>

<RequestExample>
```bash cURL
curl https://api.mirofish.com/api/report/by-simulation/sim_abc123
```

```python Python
import requests

response = requests.get(
    "https://api.mirofish.com/api/report/by-simulation/sim_abc123"
)

if response.status_code == 200:
    report = response.json()["data"]
    print(f"Report found: {report['report_id']}")
else:
    print("No report found for this simulation")
```
</RequestExample>

<ResponseExample>
```json Response
{
  "success": true,
  "has_report": true,
  "data": {
    "report_id": "report_f8a3c91b2e4d",
    "simulation_id": "sim_abc123",
    "status": "completed",
    "outline": { /* ... */ },
    "markdown_content": "# Report Title\n\n...",
    "created_at": "2025-12-09T14:15:30.123456",
    "completed_at": "2025-12-09T14:28:45.789012"
  }
}
```

```json Not Found
{
  "success": false,
  "error": "该模拟暂无报告: sim_abc123",
  "has_report": false
}
```
</ResponseExample>

## List Reports

### GET /api/report/list

List all reports with optional filtering.

**Query Parameters**

<ParamField query="simulation_id" type="string">
  Filter by simulation ID (optional)
</ParamField>

<ParamField query="limit" type="number" default={50}>
  Maximum number of reports to return
</ParamField>

**Response**

<ResponseField name="success" type="boolean">
  Whether the request was successful
</ResponseField>

<ResponseField name="data" type="array">
  Array of report objects (same structure as single report)
</ResponseField>

<ResponseField name="count" type="number">
  Number of reports returned
</ResponseField>

<RequestExample>
```bash cURL
curl https://api.mirofish.com/api/report/list?limit=10
```

```python Python
import requests

response = requests.get(
    "https://api.mirofish.com/api/report/list",
    params={
        "simulation_id": "sim_abc123",  # Optional filter
        "limit": 20
    }
)

reports = response.json()["data"]
for report in reports:
    print(f"{report['report_id']}: {report['outline']['title']}")
```
</RequestExample>

## Download Report

### GET /api/report/{report_id}/download

Download the complete report as a Markdown file.

**Path Parameters**

<ParamField path="report_id" type="string" required>
  The report ID
</ParamField>

**Response**

Returns a Markdown file with `Content-Type: text/markdown` and filename `{report_id}.md`.

<RequestExample>
```bash cURL
curl -O -J https://api.mirofish.com/api/report/report_f8a3c91b2e4d/download
```

```python Python
import requests

response = requests.get(
    "https://api.mirofish.com/api/report/report_f8a3c91b2e4d/download"
)

with open("report.md", "wb") as f:
    f.write(response.content)
    
print("Report downloaded successfully")
```

```javascript JavaScript
fetch('https://api.mirofish.com/api/report/report_f8a3c91b2e4d/download')
  .then(response => response.blob())
  .then(blob => {
    const url = window.URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = 'report.md';
    a.click();
  });
```
</RequestExample>

## Delete Report

### DELETE /api/report/{report_id}

Permanently delete a report and all associated files.

**Path Parameters**

<ParamField path="report_id" type="string" required>
  The report ID to delete
</ParamField>

<Warning>
This operation is irreversible. All report data including sections, logs, and metadata will be permanently deleted.
</Warning>

**Response**

<ResponseField name="success" type="boolean">
  Whether the deletion was successful
</ResponseField>

<ResponseField name="message" type="string">
  Confirmation message
</ResponseField>

<RequestExample>
```bash cURL
curl -X DELETE https://api.mirofish.com/api/report/report_f8a3c91b2e4d
```

```python Python
import requests

response = requests.delete(
    "https://api.mirofish.com/api/report/report_f8a3c91b2e4d"
)

if response.json()["success"]:
    print("Report deleted successfully")
else:
    print(f"Error: {response.json()['error']}")
```
</RequestExample>

<ResponseExample>
```json Success
{
  "success": true,
  "message": "报告已删除: report_f8a3c91b2e4d"
}
```

```json Not Found
{
  "success": false,
  "error": "报告不存在: report_f8a3c91b2e4d"
}
```
</ResponseExample>

## Report Status Check

### GET /api/report/check/{simulation_id}

Check if a simulation has a report and whether the Interview feature is unlocked.

**Path Parameters**

<ParamField path="simulation_id" type="string" required>
  The simulation ID to check
</ParamField>

<Info>
The Interview feature is only unlocked when a report is completed. This endpoint helps determine UI state.
</Info>

**Response**

<ResponseField name="success" type="boolean">
  Whether the request was successful
</ResponseField>

<ResponseField name="data" type="object">
  <ResponseField name="simulation_id" type="string">
    The simulation ID
  </ResponseField>
  
  <ResponseField name="has_report" type="boolean">
    Whether a report exists for this simulation
  </ResponseField>
  
  <ResponseField name="report_status" type="string">
    Report status if exists (null otherwise)
  </ResponseField>
  
  <ResponseField name="report_id" type="string">
    Report ID if exists (null otherwise)
  </ResponseField>
  
  <ResponseField name="interview_unlocked" type="boolean">
    Whether the Interview feature is unlocked (true only if report is completed)
  </ResponseField>
</ResponseField>

<RequestExample>
```bash cURL
curl https://api.mirofish.com/api/report/check/sim_abc123
```

```python Python
import requests

response = requests.get(
    "https://api.mirofish.com/api/report/check/sim_abc123"
)

data = response.json()["data"]
if data["interview_unlocked"]:
    print("Interview feature is available!")
else:
    print(f"Report status: {data['report_status']}")
```
</RequestExample>

<ResponseExample>
```json Completed
{
  "success": true,
  "data": {
    "simulation_id": "sim_abc123",
    "has_report": true,
    "report_status": "completed",
    "report_id": "report_f8a3c91b2e4d",
    "interview_unlocked": true
  }
}
```

```json Generating
{
  "success": true,
  "data": {
    "simulation_id": "sim_abc123",
    "has_report": true,
    "report_status": "generating",
    "report_id": "report_f8a3c91b2e4d",
    "interview_unlocked": false
  }
}
```

```json No Report
{
  "success": true,
  "data": {
    "simulation_id": "sim_abc123",
    "has_report": false,
    "report_status": null,
    "report_id": null,
    "interview_unlocked": false
  }
}
```
</ResponseExample>

## Response Schemas

### Report Object

```typescript
interface Report {
  report_id: string;              // Format: report_xxxxxxxxxxxx
  simulation_id: string;          // Associated simulation
  graph_id: string;               // Knowledge graph ID
  simulation_requirement: string; // Original simulation requirement
  status: 'pending' | 'planning' | 'generating' | 'completed' | 'failed';
  outline: ReportOutline | null;
  markdown_content: string;       // Complete Markdown report
  created_at: string;             // ISO 8601 timestamp
  completed_at: string | null;    // ISO 8601 timestamp
  error: string | null;           // Error message if failed
}

interface ReportOutline {
  title: string;                  // Report title
  summary: string;                // One-sentence summary
  sections: ReportSection[];      // 2-5 sections
}

interface ReportSection {
  title: string;                  // Section title
  content: string;                // Markdown content
}
```

### Status Values

- **pending**: Report generation not yet started
- **planning**: AI agent is analyzing requirements and planning outline
- **generating**: AI agent is generating section content
- **completed**: Report generation finished successfully
- **failed**: Report generation failed (see `error` field)

<Note>
Reports are stored with the following file structure:

```
reports/{report_id}/
  meta.json           # Report metadata
  outline.json        # Report outline
  progress.json       # Real-time progress
  section_01.md       # Individual sections
  section_02.md
  full_report.md      # Complete report
  agent_log.jsonl     # Agent execution log
  console_log.txt     # Console output
```
</Note>
